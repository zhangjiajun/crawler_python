<em>威尼斯</em>
<img src="http://storage.travel.sina.com.cn/jingdian/201309181216173838_20130711_163609_default_20232_pic_235.jpg" width="235" height="160px" />
</a><p class="guessbg"><span>
					

	威尼斯是一座真正的水城，其历史完全受到深入陆地的海水所形成的</span></p>

<li class=" ">
<a href="/balidao-lvyou/" target="_blank">
<em>巴厘岛</em>
<img src="http://storage.travel.sina.com.cn/jingdian/20140415_150357_default_20069_pic_235.jpg" width="235" height="160px" />
</a><p class="guessbg"><span>巴厘岛，印度尼西亚的33个行政区之一，与爪哇岛仅3公里的海峡隔海相望。这座被赋予神秘色彩的岛屿，多样的地形包括丘陵、山区、崎岖的海岸线、沙滩，造就了这里既有茂密的水稻梯田也有贫瘠的火山山坡。巴厘岛面积约5630多平方公里，印度尼西亚其中一省，省会登巴萨(Denpasar)，热门的旅行区域包括南部海滩区域，如库塔、萨努尔和奴沙杜瓦，乌布及其周边地区和巴厘岛的中部山区。
面积:5632.8平方公里，人口:389.2万人，当地语言:印尼语</span></p>

<li class=" ">
<a href="/puluowangsi-lvyou/" target="_blank">
<em>普罗旺斯</em>
<img src="http://storage.travel.sina.com.cn/jingdian/201309181215515938_20130515_114708_default_20171_pic_235.jpg" width="235" height="160px" />
</a><p class="guessbg"><span>阿维尼翁Avigon(台湾称亚维农)，她是世界历史文化名城，普罗旺斯的心脏，还被称为“教皇之城”。这里还是世界著名戏剧节阿维尼翁戏剧节的举办地，每年的7月这里就成为了戏剧爱好者的圣地。阿维尼翁城东不远处，便是《山居岁月》一书的背景地――吕贝隆山区(Luberon)，其间布满各具特色的小镇和一片片的熏衣草花田；城市往西则是梵高当年居住的阿尔勒和圣雷米(Saint-Rémy-de-provence)，以及他画中描绘的向日葵，还有很多罗马遗迹，比如：加尔桥(Pontdugard)、尼姆竞技场(ArènesdeN&icirc;mes)等；城市北方则是近年葡萄酒爱好者的新宠――教皇新堡。



阿维尼翁被古城墙包围，景点集中在城墙内，步行或搭乘小火车是游览这座城市的最好方式。一定不可错过的就是教皇宫和断桥(圣贝内泽桥)，漫步街头，不同肤色、种族，打扮各异的游客与本地人成为一道独特的风景。街头人群来来往往，街角又是一番宁静祥和的气息。普罗旺斯的风情也在这个南法小城中得到了完美的诠释。



阿维尼翁地区很适合自驾游览，景色宜人，悠闲自在。而当地游览团一般的一日游行程都是前往这两地的熏衣草花田进行参观。近期很多华人游览团推荐规模最大的瓦朗索勒熏衣草花海，规模很大，距离阿维尼翁较远，走高速大约100分钟，其中大多数是黑车黑导游，选择请谨慎。
																																					当地语言:法语english</span></p>

</li>
<li class=" ">
<a href="/xinjiapo1-lvyou/" target="_blank">
<em>新加坡</em>
<img src="http://storage.travel.sina.com.cn/jingdian/201309181215104346_20130515_115846_default_20068_pic_235.jpg" width="235" height="160px" />
</a><p class="guessbg"><span>新加坡市区
是新加坡共和国首都，位于新加坡岛最南端。新加坡市是全国政治、经济、文化中心，有&ldquo;花园城市&rdquo;之称。市区珊顿道是东南亚乃至国际的金融中心，相毗连的新加坡港(吉宝港口)，也是世界上最繁忙的港口之一。在市区新加坡河口上，矗立一座乳白石的&ldquo;狮头鱼尾&rdquo;雕像，它是新加坡的象征和标志。</span></p>

</li>
<li class=" none">
<a href="/mangu-lvyou/" target="_blank">
<em>曼谷</em>
<img src="http://storage.travel.sina.com.cn/jingdian/201309181210356629_20130510_162223_default_20048_pic_235.jpg" width="235" height="160px" />
</a><p class="guessbg"><span>曼谷KrungThep的字面意思即“天使之城”，作为“千佛之国”泰国的首都，这座城市有着数之不尽的寺庙，处处可见在晨曦中行走化缘的出家人以及保持完好的佛教习俗，然而用“传统”一词来形容曼谷却是远远不够，这里大型购物商场遍布市区、塞车程度全球首屈一指、高端品牌的广告随处可见。繁忙的商业区和安静的小巷交错在一起；文化古迹和现代夜生活区域相邻；西方文化和泰国传统文化在这里相融。曼谷就像一个什锦盘，各元素五颜六色地拼凑在了一起。
曼谷位于泰国中部，毗邻泰国湾，整座城市高于海平面仅2米。气候四季如夏，每年4月份泼水节期间，气候尤其炎热，可高达40度。湄南河穿城而过，大皇宫、黎明寺、卧佛寺、考山路等著名景点则位于河滨两岸。作为泰国第一大都市，整个大曼谷市共分为50个区，共1200多万人口。曼谷的公交车系统不够完善，BTS、MRT和机场快线是出行首选。
除了参观大皇宫、卧佛寺等经典景点之外，游客们还可在Siam商圈血拼，每年6月和12月还有降价促销；加都加周末市场、Asiatique夜市则是购买手工艺品和当地特产的好地方；考山路是全球各地背包客的聚集地，也是著名的酒吧一条街。
在享受美味的泰式冬阴功、木瓜沙拉之后，品尝鲜美的椰汁，再来一个传统泰式按摩，也许你会发现所有的花费还不超过100元人民币。曼谷就是这么一个物美价廉的地方，奢简由己，不管走高端路线还是路线，在曼谷你绝对不会亏待到自己。
面积:1568.74平方公里，人口:816万人，当地语言:泰语。</span></p>

</li>
<li class=" none">
<a href="/qingmai-lvyou/" target="_blank">
<em>清迈</em>
<img src="http://storage.travel.sina.com.cn/jingdian/20140415_162638_default_20054_pic_235.jpg" width="235" height="160px" />
</a><p class="guessbg"><span>清迈是一座被称为北方玫瑰的美丽古都，也是背包客的避风港。它坐落在滨河河畔，是泰国北部最大的城市。过去曾经作为统治北泰一带的兰纳泰王国的国都而盛极一时。清迈四面环山，地理位置很好。现在市内也有超过100所的寺庙。除了令人惊叹的寺庙以外还有许多让人兴奋不已的历史景点、熙熙攘攘的市场和活跃多彩的夜生活。清迈的繁荣建立在它的手工艺品贸易上，这里生产着以传统技艺著称的手工艺品。在沿街的小店和夜市里都能找到琳琅满目的货品。这里除了充满活力的城市和夜市之外，还有残存的城墙、城门和护城河，以及许多古老的寺庙。很多外国人被这里的温和气候所吸引，而长期居住下来。近年来清迈迅速发展壮大，高档酒店和时尚餐馆如雨后春笋般遍布各地，这种新鲜的感觉和本地的特色互相交融形成了另一种风格。
清迈的字面意思是新城，横亘在泰国北部的古都，是泰国第二大城市。早在1296年由兰纳王国的曼格莱王建为当时泰国的首都，取代了之前的首都清莱，从而逐渐发展成为泰国北部王国的经济、文化和宗教中心。在曼格莱王的统治下，清迈成为小乘佛教的一个主要发源地。在这个时期及后来的提洛克国王统治时期，清迈市内修建了大量杰出的寺庙。现在清迈被称作庙宇之城，尽管面积比曼谷小得多，但可以十分自豪地说，它的寺庙之多和首都曼谷不相上下。其中大部份庙宇建于13-16世纪中期，这也是清迈最繁荣兴旺的时期。它还是泰国主要的宗教中心，现在市内的许多寺庙都是那个时候保存下来的。后来曾有一段时间被缅甸统治，缅甸人将很多寺庙进行了改建。尽管如此，清迈的佛寺建筑仍被公认是兰纳时代的缩影。寺庙的梁柱和门上都装饰有精巧的木雕。今天的古城仍然纯朴悠闲，游客不仅被它的寺庙所吸引，还被它良好舒适的购物环境、齐全的旅行设施、优质的住宿条件所吸引。
面积:40.216平方公里，人口:14.8万人，当地语言:泰语。</span></p>

</li>
<li class=" none">
<a href="/dibai-lvyou/" target="_blank">
<em>迪拜</em>
<img src="http://storage.travel.sina.com.cn/jingdian/201309181215304569_20130510_120340_default_20117_pic_235.jpg" width="235" height="160px" />
</a><p class="guessbg"><span>迪拜，有每天五次次次不落的宣礼声；迪拜，也有灯红酒绿纸醉金迷的酒吧文化。开放的西方思想混合着闭塞的伊斯兰礼教，让你仿佛身陷在一个巨大的魔方之中，等待着去发现惊喜。从世界角度讲，阿拉伯联合酋长国是一个国家，迪拜是一个城市。但从阿联酋国内角度讲，迪拜是一个酋长国，有独立的政府和酋长，相当于自成一国自我管理。迪拜同摩洛哥的卡萨布兰卡，巴西的里约热内卢，土耳其的伊斯坦布尔一样，虽然很有名，但并不是首都，同阿联酋首都阿布扎比比起来，它也只好屈居第二了。但这位嚣张的二哥却创造了很多世界之最，例如最高的塔，最大的购物中心，最大的人工岛和最大的室内滑雪场等。



摊开迪拜地图，可以把它分为左手区和右手区，你的右手是迪拜的老城区，在地图上可以看到唯一的一条内陆河，它就是迪拜的母亲河DubaiCreek(迪拜河)，迪拜河全长只有14公里，是世界上最短的河流，却有着点石成金的魔力，是现代迪拜发展的起源地。迪拜河向东延伸形成了Deira，向西延伸形成了BurDubai，这里便是迪拜的老城区，因此大部分的历史建筑、酋长故居、文化古迹和博物馆都在河的两岸。你的左手便是被称为中东曼哈顿的TradeCenter以及Jumeirah，疯狂的建筑师们在极左地区建造了一座比一座高，一座比一座怪的楼，有些依然是进行时。在地图的最上方阿拉伯海中有三个棕榈树形状的人工岛，它们从右到左依次是ThePalmDeira、ThePalmJumeirah和ThePalmJebelAli；目前为止唯一开放的是中间的ThePalmJumeirah，岛上有超豪华的亚特兰蒂斯酒店。
																					面积:491.1平方公里								人口:226.2万人								当地语言:阿拉伯语</span></p>

</li>
<li class=" none">
<a href="/pujidao-lvyou/" target="_blank">
<em>普吉岛</em>
<img src="http://storage.travel.sina.com.cn/jingdian/201309181210342745_20130510_161714_default_20047_pic_235.jpg" width="235" height="160px" />
</a><p class="guessbg"><span>普吉岛(PhuketIsland)，泰国南部岛屿，位于泰国南部马来半岛西海岸外的安达曼海(AndamanSea)。首府普吉镇地处岛的东南部，是一个大港口和商业中心。普吉岛是泰国最大的海岛，也是泰国最小的一个府。以其迷人的风光和丰富的旅游资源被称为“安达曼海上的一颗明珠”。普吉岛自然资源十分丰富，有“珍宝岛”“金银岛”的美称。主要矿产是锡，还盛产橡胶、海产和各种水果。岛上工商业、旅游业都较发达。面积:543平方公里,人口:34.9万人,当地语言:泰语。</span></p>

</li>
<li class=" none">
<a href="/jizhoudao-lvyou/" target="_blank">
<em>济州岛</em>
<img src="http://storage.travel.sina.com.cn/jingdian/201309181210151971_20130502_143336_default_20013_pic_235.jpg" width="235" height="160px" />
</a><p class="guessbg"><span>济州岛Jeju Island 过去也写做 Cheju Island，是韩国最大的岛屿，是一座典型的火山岛，世界新七大自然奇观之一。120万年前火山活动而形成，岛中央是通过火山爆发而形成的海拔1951米的韩国最高峰―――汉拿山(Mt.Halla)。海洋性气候的济州岛素有“韩国夏威夷”之称。美丽的济州岛不仅具有海岛独特的美丽风光（瀛州十景），而且还继承了古耽罗王国特别的民俗文化。就是“三多”（石多、风多、女人多）“三无”（无乞丐、无小偷、无大门）和“三丽”（民俗、水产品和传统工艺）。，贴切地反映了济州独特的自然文化景观和济州人民朴实的民情。</span></p>

</li>
