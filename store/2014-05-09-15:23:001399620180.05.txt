<p>　　来了那么多次东京，还没有好好把最时尚、最文艺、最小资的涩谷以及周边地区好好玩遍，樱花季来到东京，终于这个愿望得以实现。这个聚集东京超时尚元素的街区是重访东京最值得造访的地区之一，这就是邻接深受年轻人喜爱的涩谷的西南面，将涩谷、自由之丘、二子玉川连起来的三角地区。这里不仅汇集了今年最流行的各式服装店，能品尝到从法国归日的厨师制作的热腾腾巧克力火锅的咖啡店，还有东京最美最小资的书店茑屋书店……</p>
<p>　　这个三角地区的起点就是大家所熟悉的涩谷，三角地区途中有氛围高雅，个性时尚的的代官山，拥有众多时尚生活杂货店和让人心情舒畅的咖啡店而备受年轻人欢迎的自由之丘，城市与自然和谐统一的人气街区二子玉川，充满怀旧复古街道的三轩茶屋，最后到中目黑赏夜樱，在满目春色的樱花季，东京最优雅的时尚街区在一天内一网打尽。</p>
<p>　　“Triangle Ticket“是三角地区(见下图地图)一日乘车通票，一天之内，可以在连接涩谷站~自由之丘站~二子玉川站区间内的所有车站随意多次上下车，一天票价仅为400日元(约合24元人民币)。也就是说从涩谷站到自由之丘站(东急东横线)、二子玉川站(东急田园都市线)，以及从自由之丘站到二子玉川站(东急大井町线)之间的所有车站，一日之内都可以自由换乘。买好划算的车票，随心所欲前往三角地区，去邂逅一个不一样的东京吧！</p>
<div class="img_wrapper"><img height="559" src="http://s11.sinaimg.cn/orignal/001HvAkMgy6IAtmzARs2a&amp;690" width="603" /></div>
<p>　　在外国游客云集的涩谷，面向外国游客的观光信息咨询处就在车站地下二楼连接通道里，可以在这里获取涩谷地区免费wifi的账号，整个涩谷地区一共有5个地点可以免费获取：涩谷站旅游观光信息咨询处(地下2楼)、涩谷站综合服务中心(地下1楼)、涩谷Hikarie信息服务窗口(2楼，地下3楼)、东急百货店总店(1楼)、东急百货店东横店(1楼，5楼)和涩谷109(1楼)。</p>
<div class="img_wrapper"><img height="403" name="image_operate_17681397915996915" src="http://i1.sinaimg.cn/edu/cr/2014/0505/3520050635.jpg" width="604" /></div>
<p>　　只需护照就能拿到免费wifi的账号及密码。</p>
<div class="img_wrapper"><img height="401" name="image_operate_37401397915997659" src="http://i3.sinaimg.cn/edu/cr/2014/0505/2451065121.jpg" width="601" /></div>
&nbsp;

<p>　　在东急地铁沿线各车站都可以买到三角地区一日乘车通票，例如涩谷站售票机上，先按画面右上方的”English”按钮，然后在新画面左方的“DiscountTicket”中选取“TriangleTicket”就可以买到一日畅行三角地区的通票了，非常方便。</p>
<div class="img_wrapper"><img height="401" name="image_operate_54061397915701136" src="http://i2.sinaimg.cn/edu/cr/2014/0505/387546553.jpg" width="601" /></div>
<p>　　一天只花24人民币就能畅游涩谷周边最时尚、最文艺、最小资的三角地区啦，Let'sGo！</p>
<div class="img_wrapper"><img height="400" name="image_operate_2711397916138189" src="http://i2.sinaimg.cn/edu/cr/2014/0505/4166350922.jpg" width="599" /></div>
<p>　　东急线运营的电车分为各站停靠的普通车(各站停车)、准快车(准急)、快车(急行)和特快(特急，通勤特急)。虽然不需要额外支付特急或急行费用，但是除了普通电车每站停靠之外，其他种类的电车均不是每站停靠，所以乘坐的时候还是要看清楚哦。</p>
<div class="img_wrapper"><img name="image_operate_51771397915563285" src="http://i2.sinaimg.cn/edu/cr/2014/0505/1256976392.jpg" /></div>
<p>　　第一站是代官山，坐东横线涩谷站的下一站便是，一出站便看见一家超人气的蛋糕店，好多日本MM在排队。代官山是东京都涩谷区中高阶级的住宅区，气氛高雅，尤其受到旅居日本的外国人和年轻女性的欢迎，有很多品味个性化的服饰，餐厅，咖啡和生活杂货店。</p>
<div class="img_wrapper"><img height="401" src="http://i2.sinaimg.cn/edu/cr/2014/0505/2190877798.jpg" width="601" /></div>
<p>　　充满着散步气氛的代官山有着城市花园的气氛，它的魅力不只是在时髦的商品，欣赏路旁小景也有另一番乐趣，随意走进一条看来只是一般住宅的巷弄里，却惊喜地发现，还有些特别的店家藏身在里面。</p>
<div class="img_wrapper"><img height="401" name="image_operate_70601397915565782" src="http://i1.sinaimg.cn/edu/cr/2014/0505/102953076.jpg" width="601" /></div>
<p>　　代官山的茑屋书店大概是东京目前最好、最美的书店了，2011年12月5日开业，店名取自江户时代浮世绘画家茑屋重三郎的姓氏，这家书店店内有大量精选的图书、杂志、唱片CD、DVD影碟，还有设计品店、咖啡休息区和Lounge餐厅，是东京一个充满浓厚文艺气息的地方。</p>
<div class="img_wrapper"><img height="402" name="image_operate_15601397915570833" src="http://i0.sinaimg.cn/edu/cr/2014/0505/1037619841.jpg" width="602" /></div>
<p>　　书店中专心看书的少年。</p>
<div class="img_wrapper"><img height="449" name="image_operate_51341397915571156" src="http://i1.sinaimg.cn/edu/cr/2014/0505/926424824.jpg" width="599" /></div>
<p>　　书店门口偶遇COSPLAY。</p>
<div class="img_wrapper"><img name="image_operate_47041397915503496" src="http://i1.sinaimg.cn/edu/cr/2014/0505/3749107002.jpg" /></div>
<p>　　代官山的下一站是中目黑站，离车站不到100米的目黑川是东京大名鼎鼎的赏樱名所之一，这是一条清澈美丽的小溪，到了春天长约3.8公里800多棵樱花树齐齐开放，花与流水呼应，美不胜收。</p>
<div class="img_wrapper"><img height="399" name="image_operate_53191397915571873" src="http://i3.sinaimg.cn/edu/cr/2014/0505/4265564416.jpg" width="598" /></div>
<div class="img_wrapper"><br /></div>
<div class="img_wrapper"><img height="401" name="image_operate_75591397915573207" src="http://i1.sinaimg.cn/edu/cr/2014/0505/1827715769.jpg" width="601" /></div>
<p>　　樱花绽放时的美丽令人赞叹，凋落后的景致同样令人叹为观止。</p>
<div class="img_wrapper"><img height="401" name="image_operate_97491397915573656" src="http://i1.sinaimg.cn/edu/cr/2014/0505/2554678763.jpg" width="601" /></div>
<p>　　目黑川边有很多非常人气的餐厅，樱花季节的午餐和晚餐都是需要提前很久预定的。</p>
<div class="img_wrapper"><img height="401" name="image_operate_58711397915573845" src="http://i1.sinaimg.cn/edu/cr/2014/0505/2861606538.jpg" width="601" /></div>
&nbsp;

<p>　　我们坐的是店外露台的位置，一边享受美食，一边赏樱，别有一番情调。</p>
<div class="img_wrapper"><img name="image_operate_26091397915577971" src="http://i0.sinaimg.cn/edu/cr/2014/0505/2007336298.jpg" /></div>
<p>　　看着来来往往赏樱的人也十分有趣。</p>
<div class="img_wrapper"><img height="399" name="image_operate_56721397915576355" src="http://i3.sinaimg.cn/edu/cr/2014/0505/766985662.jpg" width="598" /></div>
<p>　　涩谷地区的三角地带中我最喜欢的就是自由之丘，如果从涩谷站坐东横线只要8分钟就可以到达。</p>
<div class="img_wrapper"><img height="398" name="image_operate_10561397915578988" src="http://i1.sinaimg.cn/edu/cr/2014/0505/2537025535.jpg" width="596" /></div>
<p>　　自由之丘是一处安静祥和的街区，舒心惬意的休闲之地，特别是个性十足的商铺和味美天香的甜品店给我留下了非常深刻的印象。</p>
<div class="img_wrapper"><img height="397" src="http://i3.sinaimg.cn/edu/cr/2014/0505/1977816237.jpg" width="595" /></div>
<div class="img_wrapper"><br /></div>
<div class="img_wrapper"><img name="image_operate_63561397915581403" src="http://i3.sinaimg.cn/edu/cr/2014/0505/340480112.jpg" /></div>
<p>　　我那么喜欢自由之丘也是因为我在这里找到了最美的一家咖啡店——J.S.Pancake Cafe。一楼是一家卖衣服的小店，等我们噌噌噌上了三楼，哇，这咖啡店简直了，要不要那么美啊，美到令人发指了有没有！</p>
<div class="img_wrapper"><img height="400" name="image_operate_7781397915740955" src="http://i3.sinaimg.cn/edu/cr/2014/0505/2872763507.jpg" width="599" /></div>
<div class="img_wrapper"><br /></div>
<div class="img_wrapper"><img name="image_operate_15301397915777974" src="http://s4.sinaimg.cn/orignal/001HvAkMgy6IezH4vK353&amp;690" /></div>
<p>　　季节限定的草莓饼好吃到哭.。.价格是1250日元(约合75元人民币)，巨大的一个，这才是真正的性价比。</p>
<div class="img_wrapper"><img height="400" name="image_operate_62531397915744395" src="http://i2.sinaimg.cn/edu/cr/2014/0505/4002199100.jpg" width="599" /></div>
<p>　　美貌的草莓饮料650日元(约合人民币39元)，比上海的一些咖啡店一般般的饮料还便宜.。.</p>
<div class="img_wrapper"><img name="image_operate_14421397915792677" src="http://i3.sinaimg.cn/edu/cr/2014/0505/698806198.jpg" /></div>
<p>　　另一款招牌pancake，这份量比我点的那一款还要实诚。</p>
<div class="img_wrapper"><img height="398" name="image_operate_62181397915779897" src="http://i2.sinaimg.cn/edu/cr/2014/0505/406001461.jpg" width="596" /></div>
<p>　　窗外是娇艳欲滴的樱花，台前是垂涎欲滴的甜品，这一天彻底的完美……</p>
<div class="img_wrapper"><img name="image_operate_54071397915789249" src="http://s6.sinaimg.cn/orignal/001HvAkMgy6IezGEF4p35&amp;690" /></div>
<div class="img_wrapper"><br /></div>
<div class="img_wrapper"><img height="402" name="image_operate_5461397915799249" src="http://i2.sinaimg.cn/edu/cr/2014/0505/4175661079.jpg" width="602" /></div>
<p>　　自由之丘街旁排列着许多小巧的个体店铺，是寻幽探微者散步的好去处。绿树环绕的林荫道，沿路多设有长椅，可坐在树荫下读书、聊天，度过一段小憩时光。</p>
<div class="img_wrapper"><img name="image_operate_42491397915794257" src="http://i1.sinaimg.cn/edu/cr/2014/0505/2431595484.jpg" /></div>
<p>　　trainchi是自由之丘车站附近的一个小型商业街，剪票南口出来朝西偏南方向走3分钟就到了。有几个2层单体的小建筑组成，房子的表面主要采用了银色的金属和玻璃组成，没有繁杂的立面设计，简洁明快，在这里发现很多设计概念的小店，既有杂货店也有咖啡店，很值得一逛。</p>
<div class="img_wrapper"><img height="400" src="http://i3.sinaimg.cn/edu/cr/2014/0505/2546907792.jpg" width="599" /></div>
<div class="img_wrapper"><br /></div>
<div class="img_wrapper"><img height="396" src="http://i3.sinaimg.cn/edu/cr/2014/0505/937129853.jpg" width="593" /></div>
<p>　　自由之丘也是东京周边有名的高档住宅居区，这里没有繁忙的工作气分，没有喧闹的城市街道，到了傍晚，华灯初上，一切都是静静的美好。顺便提一下，很有人气的儿童图书《窗边的小豆豆》就是以自由之丘附近为舞台的，翻开书第一页第一句就是，“第一次来“自由之丘”车站，从大井町线的电车上下来后，妈妈拉着小豆豆的手，正要走出检票口。</p>
<div class="img_wrapper"><img height="401" name="image_operate_53601397915708287" src="http://i3.sinaimg.cn/edu/cr/2014/0505/799860484.jpg" width="601" /></div>
<p>　　二子玉川站坐落在多摩川沿岸，从涩谷坐快车12分钟即到。二子玉川站周边也有很多百货购物商店，餐厅，是现在东京地区新兴的人气居住地。时间关系，我们就匆匆走了一遍，如果要好好逛的话也需要大半天的时间。</p>
<div class="img_wrapper"><img name="image_operate_88361397915732979" src="http://s15.sinaimg.cn/orignal/001HvAkMgy6IezHArtY5e&amp;690" /></div>
<p>　　覆盖并围着宽阔的二子玉川车站广场伫立的便是拥有160家商店的3栋现代化大楼——“二子玉川RISE购物中心”。值得一看的是地下1层的“东急美食秀FoodShow”。使用讲究泥土质量而培育的栗子的独创糕点店，能获知生产者的从全国各地汇集“地方食品”的精品店等，只有在二子玉川能品尝的“美食”商店汇聚一堂。</p>
<div class="img_wrapper"><img height="399" name="image_operate_11841397915733429" src="http://i0.sinaimg.cn/edu/cr/2014/0505/2543163390.jpg" width="598" /></div>
<p>　　这次没有时间好好进去逛逛RISE，下次留多点时间好好来逛逛买买吧。</p>
<div class="img_wrapper"><img height="400" name="image_operate_83401397915308608" src="http://i3.sinaimg.cn/edu/cr/2014/0505/270868008.jpg" width="599" /></div>
<p>　　到三轩茶屋站的胡萝卜大厦26楼顶楼吃晚餐，相对于东京其他市中心大楼餐厅难订位人多的情况，胡萝卜大厦不光View不错，而且晚餐套餐的价格也十分合理，一个人才100多人民币。白天天气好的话，可以同时眺望到富士山，横滨港，新宿高楼区，都心高楼区。</p>
<div class="img_wrapper"><img height="400" src="http://i0.sinaimg.cn/edu/cr/2014/0505/2245968789.jpg" width="599" /></div>
<p>　　最有趣的是，胡萝卜大厦常年有一个现场FM(调频83.4MHz)的录音室，介绍所在地世田谷区内的各类生活信息。这里现在正在做节目，场内场外互动热烈哦~~</p>
<div class="img_wrapper"><img height="399" src="http://i1.sinaimg.cn/edu/cr/2014/0505/3534498984.jpg" width="598" /></div>
<p>　　三轩茶屋地区有许多怀旧的小街道，也倍受日本演艺人士的喜爱。在这里小酌一杯，或晚饭后的散步皆佳。</p>
<div class="img_wrapper"><img height="400" src="http://i2.sinaimg.cn/edu/cr/2014/0505/1063581127.jpg" width="599" /></div>
<div class="img_wrapper"><br /></div>
<div class="img_wrapper"><img height="401" src="http://i3.sinaimg.cn/edu/cr/2014/0505/966468596.jpg" width="601" /></div>
<div class="img_wrapper"><br /></div>
<div class="img_wrapper"><img height="395" name="image_operate_49391397915971577" src="http://i0.sinaimg.cn/edu/cr/2014/0505/48424815.jpg" width="592" /></div>
<p>　　最后还去了中目黑地区看目黑川的夜樱，欣赏东京夜空中最璀璨的光雕精灵，看樱花飞舞……</p>
<div class="img_wrapper"><img height="399" name="image_operate_55431397915974253" src="http://i2.sinaimg.cn/edu/cr/2014/0505/989933171.jpg" width="598" /></div>

    function _getRandom (list, num) {
        if(list.length &lt;= 10){
            return list;
        }
        var ret = [];
        var temp_array = [];
        for (var i in list) {
            temp_array.push(list[i]);
        }
        for (var i = 0; i &lt; num; i++) {
            if (temp_array.length &gt; 0) {
                var arrIndex = Math.floor(Math.random() * temp_array.length);
                ret[i] = temp_array[arrIndex];
                temp_array.splice(arrIndex, 1);
            } else {
                break;
            }
        }
        return ret;
    }

    function _getJSONP (url, callback) {
        if (!url) {
            return;
        }
        var dateStr = (Date.parse(new Date())).toString();
        dateStr = dateStr.substring(0, 8);
        var name = 'jsonp' + dateStr;
        if (url.indexOf('?') === -1) {
            url += '?callback=' + name;
        } else {
            url += '&callback;=' + name;
        }
        // url += '&dpc;=1';
        var script = document.createElement('script');
        window[name] = function (data) {
            callback &amp;&amp; callback(data);
        }
        script.src = url;
        document.getElementsByTagName('head')[0].appendChild(script);
    }
