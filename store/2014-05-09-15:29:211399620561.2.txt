<p>　　在菲律宾毛派武装“新人民军”(NPA)即将庆祝建军纪念日之际，菲军方23日证实，菲律宾反对派——菲律宾共产党及其武装组织“新人民军”领导人、菲共主席贝尼托 蒂亚姆松22日因涉嫌谋杀等指控在菲中部宿务省被菲军警逮捕。法新社称，这一事件使菲政府与反对派重启和平谈判的前景变得更加渺茫。</p>
<p>　　菲武装部队总参谋长埃曼努尔 包蒂斯塔说，与贝尼托 蒂亚姆松(Benito Tiamzon，又译作贝尼托 狄安森——观察者网注)一同被捕的还有包括他的妻子菲共秘书长威尔玛 蒂亚姆松(Wilma Tiamzon，又译作威玛。狄安森——观察者网注)在内的菲共中央委员会6名成员。</p>
<p>　　贝尼托夫妇于22日下午3时15分左右在宿雾省阿洛根山镇(Aloguinsan)被捕，菲律宾军警情报单位及执法人员在镇上拦截到蒂亚姆松的座车，两人并未拒捕。</p>
<div class="img_wrapper"><img border="1" src="http://i0.sinaimg.cn/history/cr/2014/0324/1884873019.jpg" /></div>
<p style="text-align: center;">菲律宾共产党、新人民军领导人贝托尼夫妇被捕</p>
<p>　　包蒂斯塔称，几位菲共领导人面临谋杀、多重谋杀及谋杀未遂等一系列指控。目前，他们正按照正常程序对被捕的几位菲共领导人进行司法处理。</p>
<p>　　现年63岁的贝尼托一直被视为新人民军的核心人物，他的被捕无疑给菲共势力造成重创。包蒂斯塔表示，这次逮捕行动是国家保安单位的一大胜利。</p>
<p>　　菲共领导的政治组织全国民主阵线23日强烈谴责菲政府逮捕贝尼托 蒂亚姆松等人，并要求立即将他们无条件释放。全国民主阵线和平小组主席路易斯 哈兰多尼说，蒂亚姆松夫妇是全国民主阵线的顾问，在菲政府与全国民主阵线的和谈中执行非常重要的任务。他们受全国民主阵线与菲政府签订的《关于安全和豁免保证的联合协议》的保护，逮捕他们是违反协议的行为。</p>
<div class="img_wrapper"><img border="1" src="http://i3.sinaimg.cn/history/cr/2014/0324/3292310727.jpg" /></div>
<p style="text-align: center;">菲律宾共产党、新人民军领导人贝托尼夫妇被捕</p>
<p>　　菲律宾政府和平小组当天发表声明说，政府逮捕蒂亚姆松等人并未违反这一协议。</p>
<p>　　与此同时，菲总统发言人埃德温 拉谢尔达表示，政府军将做好应对菲共及“新人民军”报复行为的准备。</p>
<p>　　据了解，菲共的武装分支“新人民军”将于3月29日庆祝建军45周年，依照过去经验，新人民军会在建军纪念日前后，针对政府保安单位的哨站或是重要民间机构发动攻击。</p>
<div class="img_wrapper"><img border="1" src="http://i2.sinaimg.cn/history/cr/2014/0324/3867435083.jpg" /></div>
<p style="text-align: center;">菲律宾毛派“新人民军”即将迎来建军45周年纪念。</p>
<p>　　“新人民军”成立于1969年，以毛主义为指导思想，进行游击战以追求“新民主”。“新人民军”已在菲农村地区开展40多年游击战，是菲主要反政府武装之一。菲军方估计，“新人民军”目前有4000多名武装人员，在菲全国大部分省份进行武装活动。</p>
<p>　　另一方面，菲律宾政府与伊斯兰分离主义组织“摩洛伊斯兰解放阵线”(MILF)，将于27日在总统府马拉坎南宫签署和平协定。观察家认为，这次对菲共领导人的逮捕行动旨在防患于未然，阻止菲共在这些重大的日子制造骚乱。菲政府表示，虽然“新人民军”游击队员的数量从上世纪80年代末的26000人减少至如今的4000人，但依然对政府构成重大威胁。</p>
<p>　　2012年8月，菲律宾总统阿基诺三世宣布缉拿逮捕菲共成员，并悬赏560万比索(约合77完人民币)。阿基诺希望在2016年任期结束时与菲共达成和平协议。然而2013年4月份，菲政府宣布由挪威牵头的和解谈判以失败告终。</p>

<p style="text-align: center;">菲律宾毛派“新人民军”战士</p>

    function _getRandom (list, num) {
        if(list.length &lt;= 10){
            return list;
        }
        var ret = [];
        var temp_array = [];
        for (var i in list) {
            temp_array.push(list[i]);
        }
        for (var i = 0; i &lt; num; i++) {
            if (temp_array.length &gt; 0) {
                var arrIndex = Math.floor(Math.random() * temp_array.length);
                ret[i] = temp_array[arrIndex];
                temp_array.splice(arrIndex, 1);
            } else {
                break;
            }
        }
        return ret;
    }
