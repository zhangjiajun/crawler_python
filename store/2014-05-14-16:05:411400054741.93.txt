<p>　　北京商报讯(记者 马元月 叶龙招)虽停牌筹划时间长达两个月，但<span id="stock_sz300273"><a href="http://finance.sina.com.cn/realstock/company/sz300273/nc.shtml" class="keyword" target="_blank">和佳股份</a></span><span id="quote_sz300273"></span>的重组最终还是以流产告终。复牌后该股不出意料地被市场看空，连续跌停，而在今年一季度开始重仓该股的广发、兴全基金也因此不幸踏进了雷区，合计浮亏约6167.13万元。</p>
<p>　　继前日无量跌停后，昨日和佳股份开盘后依然是被直接砸在了跌停板上，截至收盘报27.42元/股，跌幅达10.01%。该股从前日复牌以来，股价已累计下跌18.99%，市值蒸发了28亿元。从公开资料上看，和佳股份此番大跌与该公司终止筹划的重组事项有直接关系。5月11日晚间，该公司发布公告称，鉴于交易对方在资产、财务管理等方面规范运作不足，未能满足上市公司的收购要求和条件，其决定终止筹划本次重大资产重组事宜。这也意味着，投资者期待了整整两个月的重组已化为泡影。</p>
<p>　　利空消息袭来，资金的疯狂抛货导致和佳股份目前的股价已下降至该股一季度时的股价下方。一季度时该股的最低价为27.7元/股，最高时为40.35元/股，明显高于昨日的27.42元/股。而这也使得在一季度重仓买进和佳股份的基金陷入尴尬的境地。</p>
<p>　　北京商报记者注意到，今年一季度，和佳股份的前十大流通股东中出现了两副新面孔，它们是<span id="stock_sh601166"><a href="http://finance.sina.com.cn/realstock/company/sh601166/nc.shtml" class="keyword" target="_blank">兴业银行</a></span><span id="quote_sh601166"></span>-兴全有机增长灵活配置混合型证券投资基金、<span id="stock_sh600036"><a href="http://finance.sina.com.cn/realstock/company/sh600036/nc.shtml" class="keyword" target="_blank">招商银行</a></span><span id="quote_sh600036"></span>-广发新经济股票型发起式证券投资基金，两者分别持股487万股、446万股，合计为933万股。若广发、兴全两只基金没有在一季度减仓或者清仓，按一季度时和佳股份的均价34.03元/股作为它们的成本价来算，截至昨日它们已合计浮亏了6167.13万元。</p>
<p>　　对于广发、兴全两只公募基金不幸踩雷遭遇的损失，一位证券分析人士分析猜测说，其实和佳股份截至2013年底股价已经处于高位，并且创业板本来就风险较高了，它们还在一季度重仓买入也有可能想赌该公司的重组，而重组的流产让很多投资者都始料未及。</p>
<p>　　虽然和佳股份已连续两日跌停，跌幅巨大，但一位市场分析人士指出，“从技术面上看，该股下跌的空间依然存在，因为这两天都是无量跌停，昨日的封单量也还很大”。也就是说，上述两只基金的亏损数额恐怕还会加大。投资者也认同这种观点，“尽管本次重组的流产对和佳股份并没有经营上的损失，但由于该股停牌时间较长，且在此期间创业板大跌了，所以该公司肯定会补跌的”。有股民在股吧中如此表示。的确，从和佳股份3月11日停牌至前日，创业板已累计跌了10.59%。</p>
<p>　　而值得注意的是，这两日以跌停价卖出和佳股份的也是以机构投资者为主。该股前日龙虎榜中显示的卖方都是机构，它们合计卖出717.87万元，几乎占当日成交额的全部，散户的抛货基本没有成交。</p>
<p>　　而在昨日，卖1和卖2也是机构席位，两家机构合计卖出了2275.86万元，占昨日总成交额的96.07%。不过，这并不意味着前述两只基金已成功出货，因为和佳股份一季报中的前十大流通股东均是机构，且持仓量巨大，均在446万-1979万股之间，市值至少达上亿元，所以两只基金并未脱逃。</p><!-- news_keyword_pub,stock,sh601166&amp;sh600036&amp;sz300273 -->

    function _getRandom (list, num) {
        if(list.length &lt;= 10){
            return list;
        }
        var ret = [];
        var temp_array = [];
        for (var i in list) {
            temp_array.push(list[i]);
        }
        for (var i = 0; i &lt; num; i++) {
            if (temp_array.length &gt; 0) {
                var arrIndex = Math.floor(Math.random() * temp_array.length);
                ret[i] = temp_array[arrIndex];
                temp_array.splice(arrIndex, 1);
            } else {
                break;
            }
        }
        return ret;
    }

    function _getJSONP (url, callback) {
        if (!url) {
            return;
        }
        var dateStr = (Date.parse(new Date())).toString();
        dateStr = dateStr.substring(0, 8);
        var name = 'jsonp' + dateStr;
        if (url.indexOf('?') === -1) {
            url += '?callback=' + name;
        } else {
            url += '&callback;=' + name;
        }
        // url += '&dpc;=1';
        var script = document.createElement('script');
        window[name] = function (data) {
            callback &amp;&amp; callback(data);
        }
        script.src = url;
        document.getElementsByTagName('head')[0].appendChild(script);
    }
