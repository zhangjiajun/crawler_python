<div class="img_wrapper"><img alt="白岩松携新作《一个人和他的时代》来沪演讲" src="http://i3.sinaimg.cn/history/2014/0508/U10663P1488DT20140508095916.jpg" title="白岩松携新作《一个人和他的时代》来沪演讲" /><span class="img_descr">白岩松携新作《一个人和他的时代》来沪演讲</span></div>
<p>　　当我们为国内每年盛产两三千部长篇小说的产量感到讶异，同时又为高质量的作品何以如此稀缺感到疑惑时，隐藏其间的另一个命题或许是，当下写作者们是不是太急于表达了？在日前上海交通大学举行的题为 “二十年，我们的时代”的主题演讲中，集记者、主持人、制片人、评论员四种角色于一身的白岩松如是感慨：当下时代里的绝大多数人，都是一有了感触就表达，从而省略了漫长的追寻过程，以致盛产寿命很短的垃圾作品。</p>
<p>　　白岩松举例表示，前两年台湾出了一部叫《他们在岛屿写作》的文学电影，里面有一个学者说了一段让他感触颇深的话。“他说，探究一个人的创作，就看三个过程。首先是起于感触，你一定是有了最初的感触，或者什么事情打动了你之后，你才会想到去写一首诗、一部小说，或是一定要去做一件事情，这才有了感触完了以后的表达。但我们不要忘了，在感触和表达之间，还有一个非常重要的词，就是追寻。也就是说，感触完了之后，你不要立即进入表达，而是要去追寻，等到有了足够漫长的追寻，觉得一切成熟了之后，才走向真正的表达。”</p>
<p>　　在演讲现场，白岩松说自己至今没有开微信，也没有上微博。他认为，足够有价值的东西一定会绕弯到他这里来。“到了我这个年龄，已经不需要心灵鸡汤了。”他以电影《辛德勒的名单》为例，在买了剧本之后，斯皮尔伯格不是有了感触就立即表达，而是沉了下来。“这部剧本一放就是10年，经过10年的追寻，他拿出了伟大的电影。”白岩松现在会这样来衡量自己，“做任何一件事情，我都会问自己，有没有这个追寻的过程？尤其在当下，如何在表达和感触之间，把追寻加进去，我觉得是当下中国，尤其是我们这个时代必须要有的东西。”</p>
<div class="img_wrapper"><img alt="《他们在岛屿写作》海报" src="http://i1.sinaimg.cn/history/2014/0508/U10663P1488DT20140508095527.jpg" title="《他们在岛屿写作》海报" /><span class="img_descr">《他们在岛屿写作》海报</span></div>
<p>　　这样的追寻也意味着，我们不必急于去“找到”那个所谓的唯一的答案。白岩松举例表示，当问到《简·爱》为何被称为是女性的第二本圣经？很多人马上就会回答说，因为这部小说写出了什么是男女平等。“这个答案看似正确，但要是接下来追问，从古到今写男女平等的作品多了，为什么独独给了《简·爱》这么高的评价？由此你就知道这个答案是老生常谈，甚至是虚伪的。然而很多时候，我们就不再追问了。从这个意义上说，我们要继续寻找第二个、第三个答案。”</p>
<p>　　在白岩松看来，追寻“第二个答案” 的过程，可能特别困难，却会给你一种幸福感。“我举个例子，大家都在说老人跌倒没人扶，是很严重的道德问题。但真是这样简单吗？要知道中国的老人最怕给自己的儿女添麻烦，当他到了一定岁数之后，摔一跤就得支付多少万的医药费，又没保险，这个时候他就觉得完了，要成为自己孩子的累赘了。这样，他会牢牢抓住向他伸出的任何一只手。由此，透过表层的道德问题这第一个答案，你会看到潜藏在后面的制度与人性问题。”</p>
<p>　　也正是这种追寻，造就了一个人的不可替代性。白岩松说，常有人问他，现在传统媒体受到新媒体冲击，做电视新闻永远赶不上网络，会不会有焦虑 。“说实话，我从来都没有焦虑。我的所有新闻，全是在别人结束的那一刻开始，因为人家提供了资讯，我在这个基础上做出评论。这也就逼迫你寻找第二个、第三个答案。我非常感谢我们的同行经过了那么长的时间还给我们留了可以吃的饭。”</p>
<div class="img_wrapper"><img alt="《简爱》封面" src="http://i2.sinaimg.cn/history/2014/0508/U10663P1488DT20140508095700.jpg" title="《简爱》封面" /><span class="img_descr">《简爱》封面</span></div>
<p>　　然而对答案的追寻，并不是说我们非要追寻所谓的“终极意义”，恰恰相反，如果不能从“意义”的追寻中解脱出来，就难有独特的创造力。白岩松说：“很多人都说自己不喜欢古典音乐，问他为什么不喜欢？他会回答说，听不懂。其实就是这个‘懂’字使得无数的中国人，没能靠近最美好的音乐。而这一个‘懂’字就是我们长期以来的教育给害的，在课堂上，我们总是被要求写各种段落大意、中心思想。这样的教育，让中国人靠不近音乐，也欣赏不了舞蹈。”</p>
<p>　　某种意义上，对于“意义”的格外讲究，也毁掉了中国人人生的无数趣味。白岩松说：“中国人追求意义，很大程度上就是追求成功。当你把成功当成了最重要的标准，陪伴它的就是不择手段，厚黑学也应运而生。”在白岩松看来，中国人对漂亮地输缺少应有的尊重。“2012年伦敦奥运会，在回答体育如何激励新的一代时，奥组委委员回答说，体育会教给孩子们如何去赢，同时教会孩子如何体面并且有尊严地输。这句话说得太棒了。你想，很多时候没有输你不会想到去突破。李安正是在他过得很不如意的时候，看到了《卧虎藏龙》的原著，而正是这部电影成就了他的辉煌。”</p>
<p>　　在白岩松看来，中国人相当多的不幸福感，并不是因自己缺什么而造成的，而是看到别人比自己多了什么造成的。所以，整个社会都弥漫着一种焦虑的氛围。“我经常听说的一句话，坚持就是胜利。胜利成了最后的标准。实际上，相当多的时候坚持就是失败，因为坚持往往意味着你不好奇了，你只不过是为坚持而坚持了。但真正的幸福感，恰恰在于你对过程和细节的好奇和关注。所以，我们要学会拥有一种好奇，用好奇而不是用‘坚持’，去面对一切变动和未知。”</p>

    function _getRandom (list, num) {
        if(list.length &lt;= 10){
            return list;
        }
        var ret = [];
        var temp_array = [];
        for (var i in list) {
            temp_array.push(list[i]);
        }
        for (var i = 0; i &lt; num; i++) {
            if (temp_array.length &gt; 0) {
                var arrIndex = Math.floor(Math.random() * temp_array.length);
                ret[i] = temp_array[arrIndex];
                temp_array.splice(arrIndex, 1);

                break;
            }
        }
