<p>早就有想法把博客每天的访问流量记下来，刚好现在申请了GAE的应用，又开始学Python，正好拿这个练手。打算先利用Python把访问记录保存在本地，熟悉之后可以部署到GAE，利用GAE提供的cron就可以每天更近访问流量了。OK，开始~</p>
<p>首先是简单的网页抓取程序：</p>
<pre class="python" name="code">import sys, urllib2

req = urllib2.Request(&quot;http://blog.csdn.net/nevasun&quot;)
fd = urllib2.urlopen(req)
while True:
    data = fd.read(1024)
    if not len(data):
        break
    sys.stdout.write(data)</pre>
<p>在终端运行提示urllib2.HTTPError: HTTP Error 403: Forbidden<img alt="难过" src="http://static.blog.csdn.net/xheditor/xheditor_emot/default/sad.gif">，怎么回事呢？</p>
<p>这是由于网站禁止爬虫，可以在请求加上头信息，伪装成浏览器访问。添加和修改：</p>
<pre class="python" name="code">headers = {'User-Agent':'Mozilla/5.0 (Windows; U; Windows NT 6.1; en-US; rv:1.9.1.6) Gecko/20091201 Firefox/3.5.6'}
req = urllib2.Request(&quot;http://blog.csdn.net/nevasun&quot;, headers=headers)</pre>
<p>再试一下，HTTP Error 403没有了，但是中文全都是乱码<img alt="快哭了" src="http://static.blog.csdn.net/xheditor/xheditor_emot/default/fastcry.gif">。又是怎么回事？</p>
<p>这是由于网站是utf-8编码的，需要转换成本地系统的编码&#26684;式：</p>
<pre class="python" name="code">import sys, urllib2


headers = {'User-Agent':'Mozilla/5.0 (Windows; U; Windows NT 6.1; en-US; rv:1.9.1.6) Gecko/20091201 Firefox/3.5.6'}
req = urllib2.Request(&quot;http://blog.csdn.net/nevasun&quot;, headers=headers)
content = urllib2.urlopen(req).read()   # UTF-8

type = sys.getfilesystemencoding()      # local encode format
print content.decode(&quot;UTF-8&quot;).encode(type)  # convert encode format</pre>
<p>OK，大功告成，可以抓取中文页面了。下一步就是在GAE上做个简单的应用了~</p>
